# 末日60秒 · 玩家指南

> 陨石坠落前你只有 **60 秒**。抓紧收拾物资、躲进避难所，然后——活过 **7 天**。
> 本指南面向玩家；数值可能随版本微调，以游戏内提示为准。

---

## 1. 这是什么模式

- 全员按 **4 人一队（一个家庭）** 分组：父亲 / 母亲 / 妹妹 / 哥哥。
- 开局 **60 秒准备**：全家在住宅里抢物资，时间一到进入避难所，末日降临。
- 之后是 **7 个游戏日**（每天约 9.5 分钟）。队伍活到最后即获胜。
- 你要对抗的不是别人，是**饥饿、口渴、疯狂、污染、疾病**——当然，别的家庭也未必友善。

## 2. 快速上手：开局 60 秒

1. 开局你会出现在自家住宅，头顶显示你的**家庭身份**。
2. 抓紧搜刮：**父亲可携带 8 件**，其他成员每人 **2 件**。
3. 拿满了就去门口：**潜行 + 右键家门 = 快速存入队伍库存**（也可右键门走菜单）。
4. 优先级参考：**水 > 食物 > 药品 > 武器 > 材料（废料等）**。
5. 倒计时结束自动进入避难所——没存进去的东西不会跟着你走。

## 3. 一天的时间轴

每个游戏日约 9.5 分钟，分四段（有末日时钟或者在避难所里时，HUD 会显示时段与倒计时）：

| 时段 | 时长 | 要点 |
| --- | --- | --- |
| 清晨 | 1 分钟 | **禁止 PvP**；家里若还有黑暗角落，全队理智 -15——记得点灯 |
| 白天 | 6 分钟 | 外出探索、制作、交易的主要时间 |
| 晚上 | 2.5 分钟 | 危险时段：黑暗处刷低语怪；探索区可能出现**夜袭者**冲击家门 |
| 睡觉（晚上最后 45 秒） | 45 秒 | **回家上床**：睡觉持续回血；不在床上/在户外会扣血还可能生病 |

- 前两天所有状态消耗减半，是发展期；**第 3 天起 PvP 全面开放**，出门小心。
- 手持**末日时钟**或待在避难所可随时看时间；**收音机**能播报当前事件。

## 4. 五个状态值（左侧状态栏）

| 状态 | 归零/满会怎样 | 怎么恢复 / 预防 |
| --- | --- | --- |
| 饥饿 | 归零持续掉健康 | 任何食物都能吃（罐头/军粮/泡面……营养越高回得越多） |
| 口渴 | 归零持续掉健康 | 三档水（小/中/大瓶）、净水袋、果汁等 |
| 理智 san | 归零一段时间后**变成怪物** | 娱乐物品（收音机等，周围队友一起回）、别盯着尸体看、夜里带**手电筒** |
| 污染 | **≥70 开始缓慢掉血，满值更快**且概率生病、行动迟缓 | 防化服减半累积；**洗澡器**每天一次消耗小瓶水洗掉一半；污雨带伞、浓烟戴防毒面具 |
| 健康 | 归零=倒地/死亡 | 睡觉回血、医疗包；它是你的真正生命值 |

- **生病**：户外过夜、污染满等会生病——发烧持续掉血直到死，**必须吃药**（抗生素/药品）治愈，还会传染身边的人。
- **倒地**：健康归零先倒地，倒地后血量持续流失（约半分钟）；**队友贴身持续救援 15 秒**可救起；被补一枪则直接处决。同一天第二次倒地=直接死亡。
- 状态过低时屏幕会红闪警示；污染只在**满值**时闪红。

## 5. 探索与物资

- **家门是交互中枢**：右键门选择「外出探索 / 门外事件 / 拜访别队 / 返回」。
- 出门后 **隐身 15 秒**，落在本队对应的探索区入口。
- **物资箱**：右键开始 **3 秒搜刮**（离开 3 格会中断），每天每箱可搜一次，物资每日刷新。
- 探索区是**全服共用**的野外——会遇到其他家庭，白天（第 3 天起）与夜晚都可能开战。
- **返回住所有冷却**：白天 45 秒、**晚上只要 5 秒**（夜里危险，跑得快才活得久）。
- 留意**空投**：全服广播坐标，一箱高价值物资，先到先得。

## 6. 守住家门（夜袭与攻防）

- 部分地图夜晚会有**夜袭者**在探索区你家门外聚集并冲击家门（数量随天数增加），清晨消散。
- **门耐久 100**：怪物每秒啃门；手持**木板右键门 +10**、**铁锭 +25**（修理包 +50）。
- 铁锭加固每 3 次**门升 1 级**（最高 3 级）——撬锁工具等级不够就撬不开你家。
- **路障**（木/重型/钢筋）会被优先攻击，**尖刺陷阱**反伤减速，**警报器**当晚少刷一只，**诱饵**能把一半夜袭者引去别家。
- **门被攻破**：全队进入"室外状态"（消耗翻倍、无法睡觉回血），怪物涌入避难所——赶紧修门！
- 防人也一样：别的玩家可以用**撬棍/开锁器**闯入你家；**门锁**能抵挡一天，**门陷阱**会在有人开锁进来时报警。

## 7. 制作与科技

- **研究台**打开**科技树**，用**废料**解锁，全队共享。科技卡片和悬停提示会直接列出**该科技解锁的物品**。废料主要来自探索和击杀夜袭者。
- **合成台五件套**：工作台（工具/建筑/基础）、灶台（食物药品）、净化台（净水）、**裁缝台**（护甲/背包等装备；织布机等价）、**军械台**（近战武器/枪械/弹药/投掷物；锻造台/砂轮/铁砧等价）；家里没有家具也能自建携带版（白色混凝土标记上放置，扳手拆回）。裁缝台/军械台的携带版在工作台合成。
- **合成界面**：物品按网格罗列，顶部有**分类标签**（工具/建筑/武器/护甲/医疗/食物/娱乐）和**搜索框**；**可合成的物品自动置顶**（金框），缺料变灰、未解锁科技的加锁标。点击物品在右侧查看**介绍与材料清单**，支持 合成×1 / ×5，选中后 Shift+点击 快速合成。
- **电力**：发电机烧废料/煤（每份 90 秒），电池=3 份燃料，太阳能板白天供电；部分高级配方（狙击枪、夜视镜等）需要**供电中**才能合成，电灯还能照亮防低语怪。
- **图纸**：免费解锁一个随机可用科技，捡到就是赚到。
- **培育箱**：种下种子约 4 分钟成熟，收获 2~3 份蔬菜，有几率返还种子——稳定粮源。

## 8. 游戏币、商店与背包

- 避难所里的**电脑**每分钟轮换一个小游戏：**刷新后 20 秒内完成**才发游戏币（超时白玩）。
- 游戏币是**个人的**（不与队伍共享），可在**售货机**换物资，或抽奖机碰运气。
- **实体游戏币**：按 E 打开背包点「**兑换实体币**」把余额兑成实体币（1 枚 = 1 币）——丢给队友或交易给别队就是转账；右键实体币存回自己余额。
- 每天自动发 80 金币、每分钟 +3（金币与游戏币是两种货币，金币用于职业商店）。
- **背包**：可用格子有限（快捷栏 + 第二排），E 界面下方有第二排与装备槽；布制背包（小/中/大）能扩容，防毒面具、护甲记得穿上而不是揣着。

## 9. 拜访、交易与闯入

- 右键家门选**「拜访别队」**：选目标队，对方任意成员在 30 秒内同意/拒绝。
  - **进入避难所**：传送进对方家做客（行动受限），可通过门菜单与主人**对话**。
  - **交易**：双方各自把筹码拿在**主手**，交易窗里可以**打字聊天**谈价，双方都点确认后交换主手物品。
- 想不请自来？**撬棍/开锁器**（等级要够对方门等级）可以闯入——但小心警报和陷阱。

## 10. 每日事件与天气

- **每天清晨**每队会遇到一件剧情事件：有的直接结算，有的给出**二选一抉择**（聊天栏点击选择，90 秒不选按保守项处理），还有的需要派人**外出探险**（带手电筒，45 秒后揭晓结果）。
- **环境事件**不定时出现（屏幕顶部大字提示）：
  - **污雨**：野外淋雨加速污染——带**雨伞**或防毒面具；
  - **浓烟**：不在家就持续污染+视线受阻——**防毒面具**免疫（伞没用）；
  - **寒潮**：不在家饥饿加速+行动迟缓——尽快回屋；
  - **空投**：好事，抢！
- 每天还有一期**末日日报**（聊天栏点击阅读），了解世界动态。

## 11. 觉醒职业

- 每天清晨会有 **4 名玩家随机觉醒**一个哈比职业（平民系为主，附带各自技能），每个职业整局只出现一次。
- 觉醒是白给的战力/功能，看到自己觉醒了记得看职业介绍页学技能。
- 极小概率出现**中立职业**——他们有自己的胜利条件，未必和你一条心。

## 12. 胜负条件

- **胜利**：队伍有人活过第 7 天；或提前**抵达幸存者阵营**。
- **失败**：全队死亡/变怪即出局；全服无人存活则游戏提前结束。
- 坊间传闻：末日中还存在**别的获救方式**……多留意你捡到的奇怪物品的描述。

## 13. 实用速查

| 操作 | 方法 |
| --- | --- |
| 看队伍状态（门耐久/供电/科技/我的游戏币） | `/sre:60s info` |
| 赛前组队 | `/sre:60s team`（每队最多 4 人，整队优先同家） |
| 兑换实体游戏币 | E 背包 →「兑换实体币」按钮 |
| 快速存物资（准备阶段） | 潜行 + 右键家门 |
| 区域地图 | 手持地图物品右键；**右键加标注**、Shift+右键删除 |
| 看时间 | 手持末日时钟 / 待在避难所 |
| 理智归零倒计时中自我了断保全队 | `/sre:60s sacrifice` |
| 阅读今日日报 | 聊天栏点击提醒或 `/sre:60s newspaper` |

### 给新手的五条建议

1. **水比什么都重要**——口渴掉得比饥饿快，准备阶段先拿水。
2. **前两天猛发育**：消耗减半又禁不了几次战，抓紧探索、点科技、修门。
3. **晚上别浪**：夜袭者、低语怪、其他玩家都在等你；睡觉时间一定回床上。
4. **污染要管**：看到污染过 70 就去洗澡/穿防化服，别等满了掉血才想起来。
5. **门就是命**：木板铁锭常备，路障陷阱提前放，被破门是全队最大的减员来源。

祝你活过第七天。
