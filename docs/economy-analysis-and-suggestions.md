## 二、深度诊断：当前系统的问题

### 2.1 🔴 结构性问题

#### 问题1：回合金币经济完全线性，缺乏策略深度

当前被动收入（每10秒+5金币）+ 固定击杀奖励（+100）的模式极度线性。玩家只需"存钱→买最贵的东西"即可，零决策空间。

- 10秒 = 5金币 → 1分钟 = 30金币 → 一把左轮(285) ≈ 等待9.5分钟
- 1次击杀 = 20次被动收入 = 200秒等待
- 没有机会成本、没有投资风险、没有时效性压力

**症状**: 高端局老玩家完全掌握最优购买路径，经济系统沦为机械操作。

#### 问题2：局内局外经济完全割裂

回合金币和皮肤金币之间唯一的桥梁是 `/sre:loot coin2lottery` 命令——需要通过手动命令将皮肤金币转为抽奖次数。局内表现出色不会影响局外成长，局外氪金（皮肤）不影响局内平衡——这本身是好的设计原则，但缺乏"跨局成就感传递"。

#### 问题3：售货机系统严重欠开发

售货机目前是纯管理员工具（权限等级3），玩家无法：
- 自己放置/经营售货机
- 动态定价
- 补货
- 竞价

这导致售货机沦为"静态地图装饰"，而非动态经济节点。

#### 问题4：信息具有经济价值但没有被交易

在社交推理游戏中，**信息是最稀缺的资源**。知道谁是杀手、知道杀手买了什么装备——这些都具备巨大的经济价值，但目前系统完全没利用这一点。

### 2.2 🟡 平衡性问题

#### 问题5：武器性价比倒挂

| 武器 | 价格 | DPS/击杀效率 | 性价比判断 |
|------|------|-------------|-----------|
| 小刀 | 130 | 近战一击必杀 | ⭐⭐⭐⭐⭐ 极高 |
| 左轮 | 285 | 远程一击 | ⭐⭐⭐ 中等 |
| 霰弹枪 | 300 | 近战爆发 | ⭐⭐ 偏低 |
| 手榴弹 | 330 | AOE但300s冷却 | ⭐⭐ 偏低 |
| 精神强化 | 400 | 短暂爆发+1护盾 | ⭐⭐ 偏低 |

**老玩家共识**: 小刀+停电是最优解(270金币)，远胜于单买手榴弹(330)。

#### 问题6：工具类物品的被动收入压力

便签(10)、撬棍(35)等工具非常便宜，但开门器(80)和裹尸袋(100)相对昂贵。这导致工具使用的高度同质化——玩家几乎只买便宜工具。

#### 问题7：抽卡系统经济循环不完整

648皮肤金币一抽，但皮肤金币主要来源是通行证和任务。对轻中度玩家来说，攒648金币需要数天——缺乏"小额抽卡"选项和小保底机制。

### 2.3 🟢 被忽视的潜力

- **阵营经济差异**: 平民阵营完全没有经济系统，只能捡地图物品
- **经济胜利条件**: 没有任何通过经济获胜的途径
- **动态事件经济**: 没有"限时特价""通货膨胀""通货紧缩"等动态
- **社交经济**: 没有赠与、借贷、投资机制

## 三、创造性改革方案（6大方向）

### 🎯 方向一：局内动态经济系统

#### 1.1 浮动定价机制
> **核心理念**: 同一件商品在同一局内不同时间点价格不同。

```
实现方案:
- 首购折扣: 每件商品首次购买享受20%折扣
- 限时促销: 游戏进行到5分钟时随机3件商品7折，持续60秒
- 稀缺涨价: 每当有玩家购买某件商品，该商品价格上涨15%
- 死亡清算: 每当有玩家死亡，所有武器降价10%（恐慌性促销）
```

**代码实现路径**: 在 `SREPlayerShopComponent.tryBuy()` 中增加全局价格修正因子，由 `SREGameWorldComponent` 管理。

#### 1.2 经济生态位分化

| 阵营 | 经济特性 |
|------|----------|
| **杀手** | 标准商店 + 黑市（见方向三）|
| **平民** | 新增加：信息商店（购买线索/监控录像/足迹追踪）|
| **中立** | 新增加：对冲商店（双向押注双方胜负，赢取倍率奖励）|

#### 1.3 经济事件系统

```
"列车到站"事件: 每5分钟随机触发一次
  - 补给站: 所有商品8折，持续30秒
  - 安检升级: 武器类商品不可购买，持续60秒
  - 乘务促销: 随机赠送所有玩家一件10金币以下的工具
  - 通货膨胀: 所有价格x1.5，但击杀奖励x2，持续90秒
```

### 🎯 方向二：信息经济与情报交易

> **核心理念**: 在谋杀之谜中，信息本身是硬通货。

#### 2.1 情报商人角色

```
新中立角色: "情报贩子" (Info Broker)
- 可以花费金币"探查"目标玩家（冷却90秒）
  - 10金币: 获知目标本局是否购买过武器
  - 25金币: 获知目标当前是否持有武器
  - 50金币: 获知目标的确切角色类型（非具体角色名）
- 可以通过"情报终端"将信息出售给其他玩家
  - 买家支付情报贩子设定的价格
  - 情报贩子获得80%，系统抽税20%
```

#### 2.2 监控系统商品化

```
商店新商品:
- 监控录像带 (80金币): 查看指定区域过去15秒内经过的玩家列表
- 足迹追踪器 (60金币): 显示目标玩家最近30秒的行动轨迹（持续10秒）
- 身份扫描仪 (120金币): 高亮显示所有曾与你擦肩而过的玩家的阵营颜色（3秒）
```

#### 2.3 信息悬赏板

```
公共悬赏系统:
- 任何玩家可以发起悬赏: "提供杀手身份的可靠证据 → 悬赏50金币"
- 提交证据后，发起者确认 → 赏金自动发放
- 虚假情报 → 提交者罚款赏金的50%
- 悬赏板全局可见，促进社交互动
```

### 🎯 方向三：黑市与高风险高回报

> **核心理念**: 提供一个更强大但更危险的购买渠道。

#### 3.1 黑市入口

```
黑市仅在特定条件下出现:
- 地图随机刷新"黑市商人NPC"（每局1-2个位置）
- 停电期间黑市商品价格-30%
- 需要"暗号"才能访问（通过完成地图任务获取）
```

#### 3.2 黑市专属商品

| 商品 | 价格 | 效果 | 风险 |
|------|------|------|------|
| 消音手枪 | 350 | 击杀不触发死亡公告 | 购买时广播"有人进入了黑市" |
| 伪装斗篷 | 200 | 假装成平民外观15秒 | 被击杀时掉落，暴露身份 |
| 毒针 | 250 | 延迟60秒死亡 | 如果目标在死亡前完成任务，杀手暴露 |
| 替身娃娃 | 400 | 抵消一次击杀 | 购买后位置被全局标记5秒 |
| 电磁脉冲 | 300 | 30秒内所有监控/追踪器失效 | 所有人收到"EMP激活"警告 |

#### 3.3 风险机制

```
黑市信用系统:
- 每次黑市购买积累"嫌疑值"
- 嫌疑值>阈值: 你在地图上间歇性发光（平民可见）
- 嫌疑值极高: 全局公告"某玩家有重大嫌疑"
- 嫌疑值每60秒自然衰减1点
```

### 🎯 方向四：玩家经济互动

#### 4.1 玩家间交易

```
交易机制:
- 在指定安全区域内可以发起交易请求
- 交易内容: 金币、道具、信息
- 双方确认后成交
- 系统抽取5%交易税（防止洗钱）

限制:
- 杀手不能与杀手交易（防止组队垄断）
- 交易双方5秒内不能攻击对方（安全交易窗口）
- 单局单人最多发起3次交易
```

#### 4.2 赏金猎人系统

```
新中立角色或通用机制:
- 某玩家击杀过多 → 自动生成赏金
- 击杀赏金目标 → 获得额外金币
- 赏金金额 = 基础100 + 目标累计击杀数×50
- 赏金最高500金币封顶
```

#### 4.3 保险系统

```
商店新工具:
- 人寿保险 (150金币): 如果你在本局被击杀，你的队友获得200金币补偿
- 财产保险 (80金币): 死后不掉落已购买但未使用的道具
- 复仇基金 (200金币): 击杀你的玩家会被自动悬赏300金币
```

### 🎯 方向五：经济胜利与战略深度

#### 5.1 经济胜利条件（新模式）

```
"资本博弈"模式:
- 游戏时长固定10分钟
- 胜利条件:
  - 杀手: 击杀所有平民 OR 个人资产达到2000金币
  - 平民: 完成所有任务 OR 阵营总资产达到5000金币
  - 中立: 个人资产达到3000金币（独立胜利）

- 新增经济机制:
  - 列车股票: 4支虚拟股票，价格每30秒随机波动
  - 投资终端: 可以买入/卖出股票
  - 股息: 持有股票每60秒获得分红
  - 内幕消息: 完成地图任务获得"股票预测"线索
```

#### 5.2 列车股票系统

```
4支概念股:
- "星穹能源" (Energy) - 与击杀数正相关（越混乱越涨）
- "平安保险" (Safety) - 与存活人数正相关（越和平越涨）
- "开拓建设" (Build) - 与任务完成数正相关
- "量子科技" (Quantum) - 纯随机波动（高风险高回报）

交互:
- 地图内放置"股票终端"方块
- 买入/卖出实时结算
- 收益直接计入回合金币
- 死后股票清零（高风险！）
```

#### 5.3 经济排行榜

```
局后结算增加:
- 本局最大资本家（金币最高）
- 最佳投资者（股票收益最高）
- 最省钱玩家（消费最少但获胜）
- 散财童子（在商店花钱最多）

以上排行榜数据计入统计，并奖励额外皮肤金币
```

### 🎯 方向六：局外经济改革

#### 6.1 皮肤金币多元化消耗

当前皮肤金币只能抽卡和解锁修复角色。建议新增：

| 新消耗途径 | 消耗 | 说明 |

#### 6.2 赛季经济重置与返利

- 皮肤金币不清零，但:

## 六、风险与注意事项

1. **经济膨胀**: 浮动定价如果波动过大，可能导致金币贬值。建议设置±50%的价格波动上限。
2. **交易欺诈**: 玩家间交易需设置严格的安全窗口和次数限制。
3. **信息泄露滥用**: 情报系统需要防止组队玩家利用信息差垄断游戏。
4. **股票赌博**: 经济胜利模式需要标记为"娱乐模式"，非竞技模式。
5. **开发成本**: 建议分阶段实施，先上线P0改动收集数据反馈。
