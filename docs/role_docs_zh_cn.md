## 占卜家

- ID: `diviner`
- Goal: 以晶球窥探生死，揭穿伪装的恶徒。

### Description

占卜家（乘客阵营）
开局获得【晶球】，右键对准尸体进入 10 秒占卜施法（期间需静止不动）。
完成后获知：死者死亡时间 + 随机一项凶手线索（三选一：①凶手存活/死亡 + 凶手名字【远距离显示???，靠近后揭示】；②凶手具体职业；③凶手全局高亮 1 秒）。

## 潜行者

- ID: `stalker`
- Goal: 在阴影中观察，汲取恐惧的力量。完成进化，狩猎所有人。

### Description

杀手阵营
一阶段--[无法购买商店，只能购买撬锁器]
1.无限奔跑，拥有一层护盾。
2.按下技能键吸取能量直到进入二阶段。

二阶段
1.狩猎匕首：获得一把刀，商店可购买第二把刀，可以放在副手，可以左键刀人，主手不拿刀时不显示副手上的刀。
左键刀人冲刺，右键蓄力使用，潜行时蓄力时间变为0.1s，蓄力满时如果前面有人自动释放，并使主手刀进入冷却。若副手刀不在冷却时，右键可以使用副手的刀进行突刺。冷却时间为10秒。
2.无法奔跑
三阶段条件：用匕首击杀一定人数，积累满能量

三阶段
解锁冲刺，不需要潜行即可快速举刀，按下Q键进行冲刺，可沿y轴冲刺，冷却2.5s，潜行者只能在地面上冲刺。
