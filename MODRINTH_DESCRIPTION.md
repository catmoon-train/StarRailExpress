# StarRail Express - 哈比列车增强模组

## 🚂 简介
StarRail Express (哈比列车) 是一个基于 Minecraft 1.21.1 Fabric 的多人社交推理游戏模组。它深度重构了经典的 "列车谋杀谜题" (TrainMurderMystery) 玩法，融合了多个知名角色模组的精华，并加入了大量原创内容。本模组完全开源免费，采用 GPL-3.0-only 许可证，致力于提供最流畅、最丰富、最具视觉冲击力的列车游戏体验。

## ✨ 核心特性

### 🎭 庞大的角色系统 (100+ 角色)
我们整合并原创了超过 100 个独特角色，涵盖四大阵营，每一局游戏都充满变数：
- **平民阵营 (Civilian)**: 约 40+ 角色。包括医生、侦探、运动员、超级明星、心理学家等，通过完成任务或协助义警获胜。
- **杀手阵营 (Killer)**: 约 25+ 角色。包括忍者、执行者、爆破手、变形者、操控者等，目标是消灭所有非杀手玩家。
- **中立阵营 (Neutral)**: 约 30+ 角色。包括赌徒、小偷、食尸鬼、年兽、黑白熊等，拥有独特的个人胜利条件。
- **义警/警长阵营 (Vigilante/Sheriff)**: 约 10+ 角色。包括巡逻者、高级义警、棒球手等，拥有击杀杀手的能力。
- **原创特色角色**: 如“愚者”（塔罗会机制）、“影隼”（潜行刺杀）、“飞行员”（空中支援）等。



### 🧩 丰富的修饰符系统 (20+ Modifiers)
为游戏增加随机性和趣味性：
- **体质类**: 巨人、矮人、轻灵、矫健。
- **经济类**: 富豪（开局多钱）、 纳税 。
- **特殊类**: 恋人（共享生命）、隐秘、偏执狂、黑白、难民、远征队。
- **任务类**: 任务大师（任务进度加速）。

### 🗺️ 快捷地图编辑与投票系统
- **可视化地图配置**: 支持 JSON 格式快速定义出生点、准备区、游戏区、房间位置及背景滚动方向。
- **地图投票 (Votemap)**: 内置 `/tmm:votemap` 命令，支持暂停、恢复和终止投票，让玩家共同决定下一张地图。
- **异步复制优化**: 大幅降低地图重置时的服务器卡顿和网络包压力，实现丝滑的回合切换。
- **海量预设地图**: 模组自带多张精心设计的列车与场景地图，并支持社区自定义地图导入。

### 💻 焕然一新的 UI 与交互体验
- **更好看的 UI**: 重写了游戏内 HUD、角色展示界面、商店界面和结算面板。采用现代化设计风格，色彩鲜明，信息层级清晰。
- **体力条显示**: 实时显示玩家的冲刺体力值，让追逐与逃生更具策略性。
- **任务点提示**: 智能任务导航系统，在屏幕上提供清晰的任务点方向指示和距离提示，告别迷路。
- **操作提示**: 在屏幕右上角友好地显示内容

### 🛠️ 强大的游戏功能与命令
- **完善的命令体系**:
  - `/tmm:money` - 管理员金钱管理。
  - `/tmm:switchmap` - 快速加载、保存、扫描和随机切换地图。
  - `/tmm:game` - 游戏流程控制（开始、结束、强制胜利）。
  - `/tmm:showStats` - 查看详细的个人与全服统计数据。
- **多样化的游戏模式**:
  - **普通谋杀 (Murder)**: 经典的杀手 vs 平民/义警玩法。
  - **赌徒模式 (Gambler)**: 赌徒在限定时间内通过博弈求生，时间到则自动淘汰。
  - **TNT 标签 (TNT Tag)**: 紧张刺激的“击鼓传花”式炸弹传递，最后持有者将被淘汰。
  - **昼夜求生 (Day Night Fight)**: 融合了昼夜循环、生存要素与阵营对抗的深度玩法。
  - **更多趣味模式**: 包括捉迷藏、狙击战、恶魔轮盘等 15+ 种特色玩法。
- **回放系统 (Replay)**: 记录关键游戏事件，支持赛后复盘精彩瞬间。

### ⚡ 性能与网络优化
- **网络包减负**: 经过不严谨但有效的测试，相比原版 Wathe/TMM，本模组的发包数量显著减少，大幅降低服务器带宽压力。
- **客户端性能优化**: 对原版的内容进行大量的优化FPS可以提高2-6倍
- **数据组件优化**: 优化了 CCA (Cardinal Components API) 的数据同步逻辑，减少不必要的网络传输。

## ⚠️ 重要说明

### 兼容性
- **不兼容其他 Wathe 扩展**: 本模组会阻断原版 Wathe 运行，**请勿**同时安装 Harpymodloader, StupidExpress, Noellesroles 等其他 Wathe 附属。
- **需要 Wathe 前置**: 仅用于基础装饰方块资源，本模组不使用其任何逻辑。
- **冲突模组**: CustomSkinLoader (可能导致皮肤加载异常)。
- **推荐模组**: Sodium (钠), Iris (光影), ModMenu, Simple Voice Chat, Cloth Config API。

### 技术特点
- 使用 Mojang Mappings 重写，代码更规范。
- 命名空间保持 `trainmurdermystery` 和 `starrailexpress` 以兼容现有地图。
- 完全独立的运行逻辑，稳定性更高。
