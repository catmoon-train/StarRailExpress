末日 60 秒生存模式 · 开场演出图片素材
============================================

把生成好的镜头图放到本目录，文件名固定如下（PNG，建议 16:9，1920x1080 或更高）：

  s1.png  —  平静黄昏小镇          (S1  4s-10s)
  s2.png  —  异常天空 / 陨石光点     (S2 10s-16s)
  s3.png  —  收音机 / 防空警报       (S3 16s-21s)
  s4.png  —  家中疯狂搜刮物资        (S4 21s-27s)
  s5.png  —  冲向地下室             (S5 27s-31s)
  s6.png  —  陨石撕裂天空（峰值）     (S6 31s-34s)
  s7.png  —  地下室微光（标题背景）    (S9 40s-45s，底部留暗部放标题字)

注意：
- 仅支持 PNG。MC 1.21.1 的 NativeImage.read 只解码 PNG；JPG 会报 "Bad PNG Signature"
  并显示紫色马赛克。若手上是 JPG，请先转成 PNG。
- 素材缺失时该镜头自动降级为纯黑（不会报错、不显示紫块），文字与音效照常播放，
  方便先跑通流程再逐张补图。
- 触发指令（客户端）： /sre:client screen intro_sixty_seconds
- 出图提示词见对话记录 / 可整理进 docs/。
